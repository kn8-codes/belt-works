# Belt.works Site Direction

Updated: 2026-05-01

## Site thesis

Belt.works is a research/software lab. It builds working software from real problems and uses the software as evidence.

## Current route map

- `/` — public front door.
- `/work` — proof and shipped/in-progress work.
- `/method` — how the lab thinks and builds.
- `/services` — active service properties.
- `/rooms` — public fleet/operator cockpit proof.
- `/contact` — bring-a-problem form.

## Properties

- `data.belt.works`

- `rooms.belt.works`

## Rules

- One public CTA path: `/contact`.
- No fake mailing list.
- No retired intake branding.
- No stale client-demo surface.
- ROOMS is proof, not decoration.
